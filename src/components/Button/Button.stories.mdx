# Button

## Overview

Buttons enable users to initiate an action or event with a single press. These actions can include submitting a form, opening a dialog, canceling an action, or deleting something.

## Usage

- Buttons can have icons on either the left, right, or both sides.
- A loading spinner should be shown when an action is submitted and is still being processed.
- Use only one primary button and any additional actions should be represented as buttons with lower emphasis.
- Use Button.Group subcomponent to group together the buttons that are related to each other.
