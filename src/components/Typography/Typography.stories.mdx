### Fonts

Aether uses Poppins as a main font and Nunito as secondary font. And these fonts will not be loaded automatically.

An example of importing Poppins and Nunito.

<Source
  language='html'
  dark
  format={true}
  code={`
    <!-- index.html -->
    <link
      href="https://fonts.googleapis.com/css2?family=Nunito:wght@200;300;400;600;700&family=Poppins:wght@100;200;300;400;500;600;700&display=swap"
      rel="stylesheet"
    />
  `}
/>

<Source
  language="css"
  dark
  format={true}
  code={`
    /** main.css */
    html,
    body {
      font-family: "Poppins", sans-serif;
    }
  `}
/>

### Changing the semantic element

You can change the underlying element for a one-off situation with the `as` prop:

<Source
  language="typescript"
  dark
  format={true}
  code={`
    // There is already an h1 in the page, let's not duplicate it.
    <Typography variant="h1" as="h2">
      h1. Heading
    </Typography>;
      `}
/>

The code above will render an `h2` element with `h1` styling.
